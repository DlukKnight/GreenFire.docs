# **Overview**

## **Description**

GreenFire's biggest strength is the ability to render fully working processes by using a single component. All the metadata gets sent by the backend definitions, and the processes need minimal configuration to work.

The supported processes are as follows:

* [**Crud Process**](crudprocess.md)
* [**Wizard**](wizard.md)
* [**Dashboard**](dashboard.md)
* [**Report Process**](reportprocess.md)
* [**Read-only Process**](readonlyprocess.md)
