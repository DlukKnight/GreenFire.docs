# **Overview**

## **Description**

Controls are visual representations of data values.

All controls are linked to a property of an entity, and most of them will allow the mutation of the values of those properties with input from the user.

Some controls can change their behavior based on values of other properties and/or conditions, as well as changing their values or their options.

Controls are meant to be used inside containers, with the [**Container Row**](../containers/layout/rownodecontainer.md) being the natural selection for the direct display.
