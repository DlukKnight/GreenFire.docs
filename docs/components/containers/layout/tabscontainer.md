# **TabsContainerComponent**

## **Description**

Groups tabs together in a single container.

[**Tabs**](tabcontainer.md) cannot be used by themselves, they always have to be grouped in a Tabs Container.

This container doesn't use any specific metadata.

## Example

=== "ASP.NET"

```csharp
TabsContainerBuilder<TestEntity> generalTabsContainer = rootLayout.AddTabs();
```
